# 파이썬 input()

사용자의 응답을 저장하는 함수.

## 문법
```python
input(prompt)
```

## 함수 반환

입력함수는 사용자가 작성한 문자열을 반환합니다.

## 함수 매개변수

이 함수는 입력전에 메시지를 작성하는 프롬프트 매개변수를 받을 수 있습니다.

## 예시들:

1. 사용자의 이름을 받는다:

```python
name = input('이름이 뭔가요? ')
```

3. 사용자의 나이를 받는다:

```python
age = int(input('몇살인가요? '))
```

입력의 응답은 항상 문자열이므로, 나이를 정수로 만들려면 int () 함수를 사용하여 유형을 문자열에서 int로 바꿔야합니다.
