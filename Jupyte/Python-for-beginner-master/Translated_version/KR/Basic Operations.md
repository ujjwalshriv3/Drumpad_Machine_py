# 2개의 숫자의 덧셈, 뺄셈, 나눗셈 및 곱셈을 보여주는 파이썬 코드
## 변수 `x`와 `y`를 각각 `10`과 `5`로 초기화
```python
x = 10
y = 5
```

## 기본 연산자의 유일한 요구사항은 연산자 기호가 사이에 있는 2개의 변수를 가지는것이다

### 덧셈 예시
```python
add = x + y
```

### 뺄셈 예시
```python
sub = x - y
```

### 나눗셈 예시
```python
div = x / y
```

### 곱셈 예시
```python
mul = x * y
```

## 결과들을 출력하기
```python
print(f'{x} 와(과) {y} 의 덧셈은 {add} 이다')
print(f'{x} 와(과) {y} 의 뺄셈은 {sub} 이다')
print(f'{x} 와(과) {y} 의 나눗셈은 {div} 이다')
print(f'{x} 와(과) {y} 의 곱셈은 {mul} 이다')
```

### Output:
    10 와(과) 5 의 덧셈은 15 이다
    10 와(과) 5 의 뺄셈은 5 이다
    10 와(과) 5 의 나눗셈은 2.0 이다
    10 와(과) 5 의 곱셈은 50 이다
